# 🤝 Contributing Guide

Thanks for your interest in contributing! 🚀

We welcome contributions from everyone.

---

## 📌 How to Contribute

1. Fork this repository
2. Add your AI tool in the correct category
3. Follow the format below
4. Submit a Pull Request

---

## 🧾 Format

Please use this format:

| Tool | Free Plan | Best For |
|------|----------|----------|
| Tool Name | Yes/No | Short description |

---

## ⚠️ Rules

- Only add useful tools
- Avoid spam or duplicate tools
- Keep formatting clean
- Make sure the tool is working

---

## 💡 Suggestions

You can also:
- Improve README
- Add new categories
- Fix mistakes

---

Thanks for contributing ❤️
